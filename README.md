# PokeTactics (back-end)
API for PokeTactics app.

## 🛠️ Technologies and Tools

This API project is built using modern .NET technologies and the following libraries:

### ⚙️ Framework
- **.NET 9** – Modern, cross-platform framework for building APIs

### 💾 Data Access / ORM
- **Entity Framework Core (EF Core)** – Object-relational mapper (ORM) for .NET
- **EF Core Design Tools** – Used for migrations and design-time support
- **Pomelo.EntityFrameworkCore.MySql** – MySQL provider for EF Core

### 🗄️ Database
- **MySQL** – Relational database used as the data store

### 🛠️ Development Tools
- **Visual Studio / Visual Studio Code** – IDE for .NET development
- **.NET CLI** – Command-line tools for building, running, and managing the project
- **Git** – Version control system

## 📘 API Documentation

This project uses **Swashbuckle** to generate OpenAPI documentation and provide an interactive Swagger UI
to explore and test the API endpoints.

- **Swashbuckle.AspNetCore (Swagger)** – OpenAPI documentation and interactive UI
